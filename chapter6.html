Tables

In Html tables are allows to arrange data into rows and columns
                    Tag	        Description
                    
                    table tag	Defines a table that is supermost tag for rows and columns
                    th tag	Defines a header cell in a table
                    tr tag	Defines a row in a table
                    td tag	Defines a cell in a table
                    caption tag	Defines a table caption
                    colgroup tag  Specifies a group of one or more columns in a table for formatting
                    col tag	Specifies column properties for each column within a colgroup tag element
                    thead tag	Groups the header content in a table that include caption,th with tr
                    tbody tag	Groups the body content in a table that include after thead to last tr 
                    tfoot tag	Groups the footer content in a table
