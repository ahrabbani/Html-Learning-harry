List


There are two types of lists are:-
  1. Unordered list
  2. Ordered list
 
    ol tag	Defines an ordered list 
    ul tag	Defines an unordered list
    li tag  Defines a list item 
    dl tag	Defines a description list
    dt tag	Defines a term in a description list
    dd tag	Describes the term in a description list

Unordered list

It is black circle in default and by css we create disc, square,circle,etc.
Ex.  
  • Math
  • Science
  • Computer

Ordered list

It is number in default and by css we create roman number,letters,etc.
Ex.  
  1. Math
  2. Science
  3. Computer

Use of dl,dt and dd


dl is description list that may be ordered list or unordered list
dt is description term that describe which is list item
dd is data description that describe the dt item
Ex.-  
Algebra
A type of mathematics in which letters and symbols are used to represent numbers
Trigonometry
A type of mathematics in which we study about angles
Chemistry
A branch of science in which we study about atoms and molecules
Physics
A branch of science in which we study about work,force,physical world
Biology
A branch of science in which we study about natural things
